هو تطبيق مكتبي مخصص لطلاب فرع TIC، يهدف إلى تسهيل الدراسة اليومية من خلال جمع أهم الأدوات الجامعية في مكان واحد.  
التطبيق يعمل بدون إنترنت، ويوفر تجربة بسيطة وسريعة وفعالة.
---
Features

1. اختبارات امتحانية تحاكي واجهة الجامعة الافتراضية
- محاكاة كاملة لواجهة الامتحان الرسمية  
- نفس الشكل وطريقة التفاعل  
- تدريب عملي يساعد الطالب على الاستعداد للامتحان بثقة

2. حساب العلامات والمحصلة تلقائيًا
- حساب علامة أي مادة  
- حساب المحصلة النهائية للفصل أو السنة  
- تحديد الفصل والسنة الدراسية الحالية  
- جميع الحسابات تتم تلقائيًا وبدقة

3. مسارات برمجية جاهزة
- عرض أهم المسارات في مجال البرمجة  
- شرح واضح لكل مسار  
- خطوات عملية للبدء في كل مسار

4. مكتبة جامعية شاملة
- روابط مهمة للجامعة الافتراضية  
- برامج يحتاجها طالب TIC  
- إيميلات الكادر الجامعي  
- جميع الموارد الأساسية في مكان واحد

---
يعمل التطبيق على أجهزة Windows 64bit
